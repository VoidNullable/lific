# opencode-lific-plans

This OpenCode plugin lives in its own repository so it can be installed with a
single command:

**https://github.com/Void-n-Null/opencode-lific-plans**

```sh
opencode plugin -g Void-n-Null/opencode-lific-plans
```

It mirrors OpenCode's `todowrite` planning into durable Lific plans, reusing the
Lific MCP connection from your OpenCode config (no extra auth). See that repo's
README for setup and details.
